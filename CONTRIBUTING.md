# Contributing

We welcome contributions to the Agency Demo!

## Development Workflow

1.  **Fork** the repo.
2.  **Clone** locally.
3.  Run `make up` to start Docker.
4.  Run `make install` to install dependencies.
5.  Create a branch `feature/my-feature`.
6.  Write code and add tests.
7.  Run `make test` and `make lint`.
8.  Push and open a **Pull Request**.

## Coding Standards

- We follow PSR-12 and Magento Coding Standards.
- All new features must include unit tests.
